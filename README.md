# Getting and Cleaning Data - Course Project

This repository contains the scripts and documentation as a requirement for the completion of Data Science track Course "Getting and Cleaning data"

The data used in this project is Human Activity Recognition Using Smartphones which is available here: http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

## Files

This repository contains the following files:

- `README.md` - which basically gives an overview of the repo and brief description of the files within it.
- `run_analysis.R` - the R script that was used to extract the required dataset. 
- `tidy_data.txt` - which is the submitted or the result data of the run_analysis.R
- `CodeBook.md` - which gives a description on the variables and the calculations made.
